## Quad Cortex Device Search

A searchable database of Quad Cortex devices from Neural DSP sourced from the official [device list](https://neuraldsp.com/device-list).

## Setup

```bash
bun install
```

## Development

```bash
bun dev
```

This starts both the API server (port 3000) and the Vite dev server (port 5173).

Open http://localhost:5173 to view the app.

## Scripts

- `bun dev` - Start development servers
- `bun build` - Build for production
- `bun start` - Start production server
- `bun scrape` - Scrape device data from Neural DSP
